拓拔野

作品：《搜神记》《蛮荒记》

性别：男

外号：神帝使者、龙神太子

三生挚爱：雨师妾（前世螭雨仙子）

红颜知己：姑射仙子蕾依丽雅、流沙仙子洛姬雅

朝露昙花，咫尺天涯， 人道是黄河十曲，毕竟东流去。 八千年玉老，一夜枯荣，问苍天此生何必？ 昨夜风吹处，落英听谁细数？ 九万里苍穹，御风弄影，谁人与共？ 千秋北斗，瑶宫寒苦，不若神仙眷侣，百年江湖。
功法

五行真气生克相化

五德之身天赋，法则载于神农所授《五行谱》内

天元诀

太古奇侠古元坎配合天元逆刃所创刀法，以凌厉刚烈，变幻莫测闻名，又称金族第一刀法。刀术是为：回风石舞、星飞天外、回风舞浪、东风西顾，其中尤以“回风石舞”最为著名

潮汐流

科汗淮所授，意气双修“意如日月，气似潮汐”。可修得“移穴走脉”之功效

三天子心法

身陷苍梧之渊时所学，主以“阴阳二炁”为根本

宇宙极光流

与青帝同陷鲲腹后观看极光所悟，根据此法可将真气化作“极光电火刀”

刹那芳华

最终奥义“花开一瞬，玉老千年”，与帝鸿决战前夕领悟

种神诀

青帝灵威仰传授

心心相印诀

火神祝融传授，用于驭兽

兵器

天元逆刃

天下第一神兵，回光三宝之一，曾为古元坎所有

无锋剑

木族七大神器之一，曾为空桑仙子之兵，神帝曰：剑无锋

珊瑚笛

科汗淮所赠，封印大荒十大凶兽“珊瑚独角兽”。须以此笛吹奏“金石裂浪曲”方可解封

定海神珠

水族神物，后作为聘礼赠与六侯爷、真珠

回光三宝

是为天元逆刃、两仪钟、十二时盘，集齐三宝，可得“回光诀”，若于其他盘古大神所留阴阳八卦链、盘古九碑各相结合，可成“回光阵”，又名“两仪八极九天十二地阵”，穿梭于万千宇宙之交点，胜过光阴，拓拔野以此阵奥妙悟出“刹那芳华”

盘古九碑

盘古大神搜集天下异金所铸之物，记载“乾坤诀”。后被盘古大神用于镇守九河
